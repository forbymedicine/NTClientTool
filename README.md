# NTClientTool

Утилита для автоматизации работы с ЭЦП в программе **NTClientSoftware**.
## Возможности

-  **Автоматическое закрытие окон** – "Клиентская программа успешна запущена", "Программа продолжит работу в фоновом режиме...", "Приложение запрашивает доступ к следующим данным" 
-  **Вывод окна выбора сертификатов на передний план** – диалоговое окно выбора сертификатов автоматически становится поверх всех окон
-  **Автоматический ввод пароля** – утилита считывает пароль для контейнера из файла конфигурации, вставляет сохраненный пароль и нажимает кнопку "OK"
-  **Пропуск окна выбора сертификата** - если у пользователя установлен один сертификат, окно выбора сертификата пропускается. Если к компьютеру подключён 1 носитель ЭЦП, утилита выбирает сертификат, в зависимости от подключенного носителя. Если стоит более 1 носителя, появляется окно выбора сертификата. При появлении окна ввода пароля ЭЦП, утилита запоминает соответствие носителя и сертификата в системе, после чего сертификат будет автоматически выбирается для данной ЭЦП (например ЭЦП AvPass S/N AVP123456 и имя контейнера "Организация ... 01_02_03_04_05_06" соответствует сертификату такому-то в системе).

## Как это работает

1. Программа ожидает запуск процесса `NTClientSoftware.exe`
2. При обнаружении процесса устанавливает хуки на события окон
3. Ставит хук на функцию запроса выбора сертификата в системе в процессе NTClientSoftware.exe
4. Отслеживает появление окон, связанных с ЭЦП
5. Вводит пароль в окно с ЭЦП

## Формат файла конфигурации

Пароли хранятся в файле `%APPDATA%\Avest\saved.txt` в формате:

```имя_контейнера:пароль:серийный номер сертификата:имя контейнера```

Например:

```AvPass S/N AVP6000123456:123456781:555555555555555555555555:Организация ... 01_02_03_04_05_06```

Вручную вводить серийный номер носителя, серийный номер сертификата и имя контейнера не обязательно — данная информация сохранится автоматически при первом появлении окна ввода пароля.
## Системные требования

- **ОС:** Windows 10/11
- **Права администратора:** не требуются

## Автозагрузка

Рекомендуется добавить программу в автозагрузку:

1. Откройте редактор реестра (`regedit.exe`)
2. Перейдите в раздел:
```HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run```
3. Создайте строковый параметр (REG_SZ):
- **Имя:** `NTClientTool`
- **Значение:** `путь_до_NTClientTool.exe`
